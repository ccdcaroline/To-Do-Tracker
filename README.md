## OVERVIEW 
This is a to-do list web app that allows users to manage their tasks in an organized and interactive way. Users can add tasks, mark them as completed, delete them, and filter between all, active, and completed tasks. The app also tracks task statistics total, active, and completed and saves all tasks using localStorage, so the list stays even after refreshing the page. It is designed to be simple, clean, and easy to use for everyday productivity.

## HOW TO USE 
There are no APIs or extra installations needed for this version since everything runs directly in the browser. 
When you open the app, type a task into the input box and click “Add Task.” Your task will appear in the list below. You can check the box next to a task to mark it as completed, or click “Delete” to remove it. Use the filter buttons all, not Completed, and completed to view specific tasks. If you want to remove all completed tasks at once, click “Clear Completed.” The stats at the top will update automatically, and your tasks will stay saved even if you refresh the page.

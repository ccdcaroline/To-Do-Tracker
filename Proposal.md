**What I'm building:** A simple web-based to-do list that lets users add, complete, and manage their tasks. 

**Who it's for / Why:** This is for students, like me, who want an easy way to keep track of assignments and daily tasks in one place. It helps stay organized and not forget things. 

**Core features:**
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Filter tasks (all, completed, not completed)
- Save tasks so they stay when you refresh the page

**What I don't know yet:** 
- How to filter and display tasks based on their status
- How to delete a specific task from a list
- How to keep track of task data using arrays or objects 
- How to create and handle a checkbox list in JavaScript
